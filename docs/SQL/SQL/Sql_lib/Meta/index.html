type catalog

Catalog represented as list of databases, there is only one catalog

Catalog is stored with this directory structure: _catalog/┌──►Catalog.meta.json │ ├──►Database1_name/┌───►Table1 │ │ │ └───►Table2 │ └──►Database2_name/┌───►Table1 │ ├───►Table2 │ └───►Table3 Catalog.meta.json contains all information about this catalog, i.e. all databases (identified by their names), their tables, headers of the tables
